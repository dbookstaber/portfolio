Open-Source Work

David Bookstaber has been writing software since the early 1990s. Most of that has been trading and analysis systems, described under quantitative finance. The repositories below are public tools and demos. His current focus is real-time market-data tooling for Excel. He explains why in Streaming Real-Time Market Data into Excel: Why RTD Beats an API for Analysts.

Selected Repositories

  • OpenAI_Assistant and OpenAI_Assist_All_Tools – Python demos of the OpenAI Assistants API: the first shows function calling, the second exercises every tool type.
  • vba-clean – removes compiled P-code from Excel workbooks (.xlsm or .xlsb) that contain VBA.
  • tax_tracker – a Python package that characterizes capital gains and qualifies dividends based on trading activity, with an example notebook.
  • vscode-outline-plus – a Visual Studio Code extension for navigating, visualizing, and managing code regions.
  • pyballistic – a fork he maintains of the py-ballisticcalc engine: a point-mass trajectory calculator supporting custom drag models.

These and other repositories are at github.com/dbookstaber.
